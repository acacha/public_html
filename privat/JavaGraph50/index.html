Java Graph Applet, 5.0

Java Graph és un projecte de l'assignatura de Matemàtica Discreta de la Universitat Oberta de Catalunya.

Aquest Applet permet representar un graf i mostrar el funcionament de diferents algorismes. En la versió actual, podeu representar deu tipus de grafs amb un nombre variable de vèrtexs (entre 3 i 15), amb pesos i sense pesos. També podeu seleccionar sis algorismes i interactuar amb els grafs. Premeu el símbol ? per obtenir-ne les instruccions.

Nota tècnica: Per a una correcta simulació s'aconsella utilitzar un navegador amb suport per a Java 1.2x. També recomanem una resolució de 1200x768.
